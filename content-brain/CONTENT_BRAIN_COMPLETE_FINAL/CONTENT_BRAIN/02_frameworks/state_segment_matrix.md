# State-Segment Matrix™ — Signal Resolution

## Definition
A model that maps the same ICP profile across different buyer states to reveal readiness variation and produce actionable targeting segments.

The core insight:

Same company profile ≠ same buyer.
Same profile + different state = different conversion probability, different message, different sales cycle.

---

## Why This Model Exists

After passing the Firmographic Gate™ and applying Psychographic Resolution™, you have a set of companies that:

- Are economically qualified
- Are experiencing some version of the problem

But they are not all the same.

One segment is in acute pain with high urgency — searching for a solution, under pressure, ready to buy.

Another segment is aware of the problem but not yet feeling urgency — curious, exploring, not prioritizing it.

A third segment is in the problem but blaming the wrong layer — frustrated, attributing it to sales or creative, not yet open to the real frame.

Targeting all three the same way produces average results across all three.

The State-Segment Matrix™ separates them so each can be addressed with the right message, at the right urgency level, with the right sales approach.

---

## The Matrix Structure

The matrix maps two axes:

**Axis 1: Firmographic Profile** (horizontal)
The company's objective characteristics — ARR range, team size, motion type, stage.

**Axis 2: Psychographic State** (vertical)
The buyer's current internal state — urgency, clarity, awareness, blame attribution.

Each cell in the matrix = a distinct targeting segment.

---

## The Five Psychographic State Buckets

### State A: Acute Pain + High Urgency
- Experiencing the problem right now
- Using complaint language ("nothing is converting," "CAC is rising and we don't know why")
- Under pressure from external factors (investors, runway, board)
- Actively looking for a solution
- **Conversion probability: Highest**
- **Sales cycle: Shortest**

### State B: Aware + Moderate Urgency
- Knows the problem exists
- Not yet in crisis
- Exploring options without commitment
- Using curiosity language ("I've been thinking about our ICP")
- **Conversion probability: Medium**
- **Sales cycle: Medium**
- **Content strategy: Increase urgency by surfacing the cost of inaction**

### State C: Misdirected Blame
- Experiencing the problem but attributing it incorrectly
- Blaming sales, creative, or lead volume
- High frustration, but not open to the real frame yet
- **Conversion probability: Medium-High once reframed**
- **Sales cycle: Longer (requires reframe first)**
- **Content strategy: Diagnostic content that reveals the hidden layer**

### State D: Unaware
- Has the firmographic profile but not yet experiencing the problem acutely
- Not in market
- **Conversion probability: Low**
- **Do not prioritize for outbound — let content reach them passively**

### State E: AI-Era Misaligned (Advanced)
- Experiencing signal degradation due to AI adoption patterns
- Traditional metrics are declining for non-traditional reasons
- Execution velocity is high but strategic direction is unclear
- **Applicable to more sophisticated scaling-stage buyers**
- **Map to the AI-Era Signal Index™ for deeper resolution**

## How to Build the Matrix

Step 1: Define your firmographic tiers (primary ICP, secondary ICP).
Step 2: Run your pipeline and target market through the Firmographic Gate™.
Step 3: Apply Market State Signals™ classification to identify psychographic state for each account.
Step 4: Place each account into the appropriate matrix cell.
Step 5: Prioritize outbound toward State A and State C (highest readiness and reframe potential).
Step 6: Use content to move State B and State D toward State A over time.

---

## Output: Actionable Targeting Segments

The matrix produces at minimum three actionable segments:

| Segment | State | Priority | Approach |
|---|---|---|---|
| Ready Now | A (Acute + High Urgency) | 1 | Direct outbound, diagnostic CTA |
| Reframe Needed | C (Misdirected Blame) | 2 | Diagnostic content, then outbound |
| Nurture | B (Aware + Moderate) | 3 | Authority content, patience |
